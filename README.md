# Crucial Web StudioThis is a repo for managing machine readable artifacts for the [Crucial Web Studio APIs](http://www.crucialwebstudio.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Crucial Web Studio APIs](http://www.crucialwebstudio.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.crucialwebstudio.com/)- [Developer](http://www.crucialwebstudio.com/)- [Twitter](https://twitter.com/CrucialApps)The [APIs.json](https://github.com/api-evangelist/crucial-web-studio/blob/master/apis.json) index provides links to all the relevant links for the Crucial Web Studio APIs, including the OpenAPI and Postman if they are present.
